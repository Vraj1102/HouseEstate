# HOW TO RUN THE PROJECT

## IMPORTANT: Run from ROOT directory, NOT from admin folder!

### Step 1: Open Terminal in ROOT directory
```
C:\Users\Vraj\Downloads\MCA\Sem-4\Real-Estate-mern-main
```

### Step 2: Start Backend (Terminal 1)
```bash
npm run dev
```

### Step 3: Start Frontend (Terminal 2)
```bash
cd client
npm run dev
```

### OR Run Both Together (Single Terminal)
```bash
npm run dev:all
```

## DO NOT run from admin folder!
The admin folder is OLD and should be ignored.
All admin functionality is integrated in the main client.
